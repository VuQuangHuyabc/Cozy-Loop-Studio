# Cozy Loop Studio LLC - E-commerce Website

A complete, responsive e-commerce website for selling premium wool scarves, built with HTML, CSS, and JavaScript.

## 🛍️ Features

### Core Functionality
- **Product Catalog**: Dynamic product grid with 9 wool scarf products
- **Shopping Cart**: Full cart functionality with add/remove/update quantity
- **Search & Filter**: Product search and category filtering
- **Responsive Design**: Works perfectly on all devices
- **Local Storage**: Cart data persists between sessions

### Pages Included
1. **Home** (`index.html`) - Hero section, featured products, company overview
2. **Products** (`products.html`) - Full product catalog with search and filters
3. **About Us** (`about.html`) - Company story, team, and values
4. **Contact Us** (`contact.html`) - Contact form, FAQ, and location information
5. **Shopping Cart** (`cart.html`) - Cart management and checkout flow
6. **Privacy Policy** (`privacy-policy.html`) - Complete privacy policy
7. **Shipping Policy** (`shipping-policy.html`) - Shipping information and policies
8. **Terms & Conditions** (`terms-conditions.html`) - Legal terms and conditions
9. **Returns & Refunds** (`returns-refunds.html`) - Return policy information

## 🔧 Customization

### Adding New Products
1. Add product images to appropriate folder
2. Update product data in `script.js`
3. The website will automatically display new products

### Modifying Styles
- Edit `styles.css` for visual changes
- CSS variables are defined at the top for easy theming
- Responsive breakpoints are clearly marked

- Contact information is centralized in the footer
